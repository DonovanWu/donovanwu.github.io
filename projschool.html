School Research Projects

  • Building simulation environment for an AI robot

    Description:

    This is a school research project in which we build a robot with artificial intelligence that provides service such as self-navigation and answering people's question. To make things convenient, we want to make it possible to test the robot's function in a computer simulation rather than having to test it in reality every time. Therefore I helped to build this simulation environment using blender game engine. Some of the components in the simulation comes from Morse simulation package.
