  • SARA Semantics Aware Robotic Assistants

    Overview:

    This is a school research project on Artificial Intelligence in CSE department. The project builds a robot that can navigate through the building by itself and finish some simple tasks people ask, such as checking if a certain person is in a certain room.

    Sub-project 1: Simulation

    The simulation uses a modified morse package and runs in blender game engine.

    Primarily worked on...

    • building a computer simulation for the robot.
    • setting up the CSE building environment in the simulation.
    • setting up human object (with elevator-calling function) in the simulation.

    Main purpose of the simulation:

    • The simulation serves as a convenient way to test out robot's function, new code, etc. so that it is not necessary to do it in reality every time. Also, the simulation makes it possible for multiple people to do robot testing at the same time because they can just do it on their own computer, while testing out in reality on actual robot only allows one person a time.

  • Nanocrafter

    Overview:

    This is a research project in Center for Game Science (CGS) at University of Washington, in which we made an educational game. Nanocrafter is a game that teaches people about DNA as well as a tool of doing DNA reaction simulation.

    Sub-project 1: Implementing helping hand's tutorial for concentration change

    Implemented new function to helping hand in game and made it able to point to + or - sign position of the widget that shows DNA concentration.

    Level designers can tell the helping hand where to go by putting its position code (updated now) in level JSON.

    Sub-project 2: VCR control of reaction playback

    Consists of both front-end and back-end work:

    • On front-end side: Designed playback button layout during reaction playback and implemented them (most of their code already existed, such as pause and rewind, so I just linked the code to playback buttons)
    • On back-end side: Modified playback animation class to take animation speed into consideration, so that fast-forward and slow-down buttons actually work. The animation speed uses a jumping or interpolating frame fashion to speed up or slow down playback animation.

    Sub-project 3: Concentration split animation

    Concentration split action in game no longer just kicks the split DNA off a little bit, but replaced by a smooth transition animation. The split DNA also cleverly finds a relatively empty space on stage and moves to that position.
